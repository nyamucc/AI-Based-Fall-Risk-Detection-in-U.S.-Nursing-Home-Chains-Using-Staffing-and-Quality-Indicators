### Project Overview
This project applies machine learning to identify nursing home chains at high risk of fall-related injuries using publicly available quality and staffing data.

### Objective
To develop an AI-based risk classification model that helps detect facilities with elevated fall risk based on operational and clinical indicators.

### Data Source
Data derived from quality reporting systems maintained by the
Centers for Medicare & Medicaid Services.

### Methods
Data cleaning and preprocessing
Feature selection (staffing, inspections, outcomes)
Binary classification using Random Forest
Feature importance analysis

### Key Findings
Lower staffing levels are associated with higher fall risk
Higher staff turnover correlates with worse outcomes
Facilities with more regulatory penalties show elevated risk
